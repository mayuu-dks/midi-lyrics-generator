MIDI Lyrics Generator

Web Component デモ

説明

MIDI Lyrics Generatorは、MIDIファイルを分析してAIを使用して歌詞を生成するツールです。このページでは、Web Component化されたバージョンをデモとして紹介しています。

注意: このツールはAnthropic Claude 3.7 Sonnet、OpenAI GPT-4o、Google Gemini 2.0などのAIサービスを利用します。APIキーがあればアプリ内で生成ができますが、なくてもプロンプトの生成とコピーが可能です。

Web Componentデモ

使い方

このWeb Componentは以下のようにあなたのウェブサイトに組み込むことができます：

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>MIDI Lyrics Generator</title>
  <script src="midi-lyrics-generator.js"></script>
</head>
<body>
  <midi-lyrics-generator></midi-lyrics-generator>
</body>
</html>
